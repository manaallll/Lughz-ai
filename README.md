LughzAI is a modern web application that combines the power of OpenAI's GPT-3.5 with an engaging user interface to deliver personalized riddles across various categories. Players can test their knowledge, earn points, compete on leaderboards, and enjoy a seamless gaming experience with beautiful animations and dynamic backgrounds.

### Key Highlights
 **AI-Generated Riddles**: Powered by OpenAI GPT-3.5 Turbo
 **Multiple Categories**: Space, Science, History, Geography, Math, Technology, and Animals
 **Scoring System**: Earn points based on performance (15 points without hints, 5 with hints)
 **Leaderboard**: Compete with other players globally
 **Secure Authentication**: JWT-based user authentication
 **Modern UI**: Built with React, Tailwind CSS, and Framer Motion

---

## Features

### Game Features
- **Dynamic Riddle Generation**: Get unique riddles generated on-demand using AI
- **Multiple Difficulty Levels**: Different categories offer varying complexity
- **Hint System**: Use hints to get help, but with reduced points
- **Real-time Scoring**: Track your progress as you play
- **Category Selection**: Choose from 7 different knowledge categories

### User Features
- **User Registration & Login**: Secure account creation and authentication
- **Profile Management**: Track your scores and progress
- **Leaderboard**: See how you rank against other players
- **Responsive Design**: Works seamlessly on desktop and mobile devices

### Technical Features
- **RESTful API**: Clean, well-documented FastAPI backend
- **Database**: SQLite database for user and score management
- **CORS Enabled**: Properly configured for frontend-backend communication
- **Error Handling**: Comprehensive error handling and validation

##  Tech Stack

### Frontend
- **React 19** - Modern UI library
- **Vite** - Fast build tool and dev server
- **React Router DOM** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Three Fiber** - 3D graphics rendering
- **Axios** - HTTP client for API calls

### Backend
- **FastAPI** - Modern Python web framework
- **SQLAlchemy** - ORM for database operations
- **SQLite** - Lightweight database
- **JWT** - JSON Web Tokens for authentication
- **Passlib** - Password hashing (bcrypt)
- **OpenAI API** - AI riddle generation
- **Pydantic** - Data validation
